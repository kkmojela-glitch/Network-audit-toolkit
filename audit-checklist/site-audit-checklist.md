# Site Audit Checklist

## Physical Inspection
- Identify all network cabinets/racks
- Check cable management quality
- Verify patch panel labeling
- Locate switches and routers

## Switch Checks
- Record switch model and serial number
- Check port status (active/unused)
- Identify uplink ports
- Check LED indicators

## Router / Firewall
- Record device details
- Check WAN/LAN connectivity
- Confirm uptime status

## Network Testing
- Ping gateway
- Test internet connectivity
- Check basic latency

## Documentation
- Take photos of equipment
- Record IP addressing scheme
- Document all findings
