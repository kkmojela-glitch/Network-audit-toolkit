# Network Site Audit Report

## Site Information
- Site Name: Example Corporate Office
- Location: Gauteng, South Africa
- Date: 2026-06-12
- Engineer: Kabelo Mojela

---

## 1. Executive Summary
A physical and logical network audit was performed on-site to evaluate infrastructure health, device inventory, and connectivity status. The network is generally stable with minor documentation and cable management issues identified.

---

## 2. Infrastructure Overview
The network consists of:
- 1 Router (ISP gateway)
- 1 Core Switch
- Multiple Access Points
- End-user devices (PCs and printers)
- Local server infrastructure

---

## 3. Findings

### 3.1 Network Devices
- Cisco switch identified in server rack
- MikroTik router handling WAN connectivity
- 1 firewall device present (basic filtering enabled)

### 3.2 Cabling
- Patch panel labeling inconsistent
- Some Ethernet cables not properly managed
- No clear cable documentation available

### 3.3 Connectivity
- Internet connectivity stable
- Internal LAN communication functional
- No major packet loss detected during testing

---

## 4. Issues Identified
- Missing structured asset labeling
- Cable management requires improvement
- No centralized documentation of IP scheme

---

## 5. Recommendations
- Implement proper cable labeling system
- Maintain updated asset inventory sheet
- Standardize IP addressing documentation
- Perform quarterly network audits

---

## 6. Conclusion
The network is operational and stable but requires improved documentation and physical organization to meet enterprise-level standards.
